# Brenden's Blog

A React blog application with a unique star-themed design.

## Getting Started

### Prerequisites

- Node.js (version 14 or higher)
- npm (comes with Node.js)

### Installation

1. Install dependencies:
```bash
npm install
```

2. Start the development server:
```bash
npm start
```

The app will open in your browser at [http://localhost:3000](http://localhost:3000)

### Available Scripts

- `npm start` - Runs the app in development mode
- `npm build` - Builds the app for production
- `npm test` - Runs the test suite
- `npm eject` - Ejects from Create React App (one-way operation)

## Features

- **Landing Page**: Eye-catching entrance with animated star graphics
- **Blog List**: Grid view of all blog posts with search functionality
- **Blog Detail**: Full blog post view with back navigation
- Responsive design that works on desktop and mobile
- Custom color scheme with banana cream and dusty olive tones
